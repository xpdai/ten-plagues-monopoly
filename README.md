# 🏜️ 出埃及大冒險 (Moses' Exodus)

A board game following Moses from the burning bush to the song by the sea —
26 stations, 2–4 players.

**▶ Play it: https://xpdai.github.io/ten-plagues-monopoly/**

- Two play modes — 遊戲軟體模式 (web mini-games) and 大地遊戲模式 (physical) —
  each with 關主模式 (vs host) or 玩家對戰模式 (designate a player to duel; winner
  may swap places).
- Software mode has interactive mini-games on every challenge tile (whack-flies,
  dance pad, typing race, dodge hail, drag across the Red Sea, paint the door,
  arrange the cloud-pillar, photo upload, and more).
- Physical mode shows the task to act out, with countdown timers and "holding a
  pose" reminders.
- 機會與命運 cards, position-swap duels, score penalties, and an exact-roll
  finish (aeroplane-chess rebound on overshoot).
- Single self-contained `index.html` — no build, no server, no dependencies.
