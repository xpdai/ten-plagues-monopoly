# 🏜️ 出埃及大冒險 (Moses' Exodus)

A cozy board game following Moses from the burning bush to the song by the sea —
26 stations, 2–4 players.

**▶ Play it: https://xpdai.github.io/ten-plagues-monopoly/**

- Two play modes — 遊戲軟體模式 (web mini-games) and 大地遊戲模式 (physical) —
  each with 關主模式 (vs host) or 玩家對戰模式 (designate a player to duel, winner
  may swap places).
- Roll exactly onto the final station to win (aeroplane-chess rebound on overshoot).
- Single self-contained `index.html` — no build, no server, no dependencies.

> Skeleton build: the mini-games are currently shown as task cards; full
> interactive mini-games are being added station by station.
