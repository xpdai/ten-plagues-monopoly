# 🏜️ Ten Plagues Monopoly / 十災大富翁

A cozy hot-seat board game for 2–4 players — race along a winding desert trail
from Goshen to the Promised Land, dodging the ten plagues with Faith, the Staff
of Moses, and a Blessing Charm.

**▶ Play it: https://xpdai.github.io/ten-plagues-monopoly/**

- Single self-contained `index.html` — no build, no server, no dependencies.
- English / 繁體中文 language toggle (top-right).
- Click any tile or player to inspect full details.

To run locally, just open `index.html` in a browser.
