# 🏜️ 出埃及大冒險 (Moses' Exodus)

A board game following Moses from the burning bush to the song by the sea —
27 stations (起點 + 26), 1–4 human players plus optional computer players.

**▶ Play it: https://xpdai.github.io/ten-plagues-monopoly/**

A single self-contained `index.html` (plus a service worker for offline) — no
build, no server, no dependencies.

## Features

- **Two play modes** — 遊戲軟體模式 (on-screen mini-games) and 大地遊戲模式
  (physical field game with act-out tasks, countdowns and "hold the pose"
  reminders).
- **Two ways to play** — 關主模式 (a referee 關主 hosts, judges, and plays
  rock-paper-scissors; a 🤖 computer player can race you) or 玩家對戰模式
  (duel a chosen player; the winner may swap places). Available in both modes.
- **Flexible line-up** — +/− steppers for human players (1–4) and computer
  players (0–4); a single player vs the 關主 is fine.
- **Race to finish** — roll to advance; exact roll to reach the goal, otherwise
  rebound off the end like aeroplane chess. 機會與命運 cards and a 追兵 setback.
- **Candy-Crush-style map** — round candy nodes on a beaded winding trail, a
  bouncing current node, a gold goal star, roadside scenery, and cleared
  stations dimmed.
- **Scripture references** — each station shows its Exodus passage (和合本).
- **Printable 關主手冊** — a one-page guide of every station's task for running
  the physical game.
- **Offline / installable (PWA)** — works after first load, installable to the
  home screen; icons fall back to inline SVG if offline.
- **Save & resume** — an unfinished game is offered for resume on next visit.
- **Sound** — synthesised background music + effects, with independent toggles
  and a music-volume slider.

Icons by [Icons8](https://icons8.com).
